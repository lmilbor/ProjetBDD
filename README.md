PROJET BDD

1/ LE SI JobOverview
Fichiers fournis :
- L'ensemble des vues en pdf Data Modeler,
- Le JOBDD.dmb et le zip contenant le reste de fichiers Data Modeler,
- La sauvegarde de la base de données JobOverview.

Les scripts à executer dans l'ordre suivant pour avoir une base :
- Le script ProcedureEtFonctionJO.sql (qui ne contient pas de fonction mais une vue et des procedures d'ailleurs) à lancer en premier,
- Le script CréationBaseJO.sql pour créer la base de données à lancer en deuxième.

Pour remplir la base :
- Le script InsertionValeurJO.sql à lancer pour remplir quelques tables (nécessaire pour lancer des tests).

Pour faire différentes requêtes de test :
- Le script ScriptTestJO.sql pour tester différentes requêtes.

2/ Requêtage sur la bas Grand-Hôtel
Fichiers fournis :
- Le script Grand-Hotel.sql ou on trouve les solutions trouvées au requêtes demandé,
- La base de données GrandHotel.bak a priori non modifié (les requêtes de modification étant précédées d'un "begin tran" et suivis d'un "rollback").
